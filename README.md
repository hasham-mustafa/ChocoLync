# ChocoLync
Production-style e-commerce application built with ASP.NET Core MVC, Razor Views, Entity Framework Core, and SQL Server. Designed using the MVC architecture and Code First approach, featuring role-based authentication, inventory management, shopping cart, order processing, and a responsive user interface.
To run this project successfully first to change the db connection in appsetting.json then run the ef commands for initial create and update database then can run the project and implement the structure accordingly.
